# Animated Portfolio Website

A deployable static portfolio for Sai Saran Tottempudi with responsive layout, animated hero canvas, reveal transitions, project visuals, counters, and a polished contact section.

## Customize

Edit these details in `index.html` when needed:

- GitHub project links
- Project names and descriptions
- Stack items
- Contact links

## Run locally

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Deploy

This is a plain static site, so you can deploy it on Netlify, Vercel, GitHub Pages, Cloudflare Pages, or any static host. Use the project folder as the publish directory.
